# X-to-Blinko

这是一个 Chrome 浏览器扩展，用于将 X.com (Twitter) 上的推文一键转发到您的 Blinko 笔记应用中。

## 功能特点

- **一键保存**：在每条推文下方添加一个 Blinko 图标按钮。
- **智能提取**：自动提取推文内容、作者、头像、时间、链接以及图片。
- **Markdown 格式**：自动将推文转换为精美的 Markdown 卡片格式保存。
- **无感交互**：保存成功后通过底部 Toast 提示，不打断浏览体验。
- **自定义配置**：支持配置您自己的 Blinko 服务器地址和 Token。

## 安装指南

1.  下载本项目代码到本地。
2.  打开 Chrome 浏览器，进入扩展程序管理页面 (`chrome://extensions/`)。
3.  开启右上角的“开发者模式”。
4.  点击“加载已解压的扩展程序”，选择本项目所在的文件夹。

## 配置说明

首次使用前，请点击浏览器右上角的 X-to-Blinko 扩展图标进行配置：

1.  **Blinko BASE_URL**: 您的 Blinko 服务器地址 (例如: `https://my-blinko.com`)。
2.  **SECRET_TOKEN**: 您的 API 密钥。

如果未配置，点击保存按钮时会弹出提示。

## 使用方法

1.  刷新 X.com 页面。
2.  在推文底部的操作栏（书签按钮左侧）会看到一个新的 Blinko 图标。
3.  点击图标，推文内容将自动发送到您的 Blinko 笔记中。

## 隐私说明

本扩展仅在您点击按钮时提取当前推文内容，并将其发送到您配置的服务器。不会收集任何其他个人信息。
